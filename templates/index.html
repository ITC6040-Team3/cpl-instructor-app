Team 3 Prototype

CPL Assistant

A prototype for certification-based course waiver intake. Students can submit required information step by step, upload supporting evidence, and help the university review team process requests more efficiently.

Open Chat Workspace Open Admin Status

What this prototype supports

  • Session-based chat workflow
  • Certification waiver intake support
  • Evidence upload and file tracking
  • AI-generated case summary
  • Structured evidence extraction
This prototype is deployed on Azure App Service with Azure SQL persistence for session, message, and evidence management.

Workspace Overview

How It Works

The assistant is designed as a guided intake workspace for certification-based waiver requests.

  • Start a new case session in the chat workspace
  • Submit course and certification information step by step
  • Upload supporting evidence tied to the current session
  • Review the generated case summary and structured evidence
